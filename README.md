# Complaint Management System

## Project Description

A web-based complaint management system that enables users to register complaints and allows administrators to efficiently track, manage, and resolve them.

---

## Features

* User registration and authentication
* Complaint submission and management
* Complaint status tracking (Pending, In Progress, Resolved)
* Admin dashboard for monitoring complaints
* Complaint categorization (Electrical, Plumbing, Mess, etc.)

---

## Database Design

Tables used in the system:

* Users / Students
* Complaints
* Departments
* Staff
* Assignments

Key DBMS concepts:

* Primary Key and Foreign Key
* One-to-Many relationships
* Normalization (1NF, 2NF, 3NF)
* CRUD operations
* JOIN queries and GROUP BY

---

## Tech Stack

* Frontend: HTML, CSS
* Backend: PHP
* Database: MySQL
* Server: XAMPP
